# <img src="public/icons/48.png" width="45" align="left"> N Yobi Char-counter

## Features
- N予備校の記述式のテストのさいに入力欄の下に文字数を出します。
- いくつかのフォーマット形式に対応しています。
- Popupでどこでも文字数をカウントできる

## 開発に貢献する
Pull RequestやIssueはウェルカムです！  
Version構成
```
Chrome: 最新版
Node: v14.17.1
```
###  デバッグ
```
yarn serve
```
のあとdistフォルダをChromeの拡張機能のページで読み込ませる
